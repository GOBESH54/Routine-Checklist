# Smart Daily Routine & Analysis Tracker

A modern, responsive React web application for tracking daily routines with automatic adaptation between College and Holiday modes.

## Features

- **Dynamic Day Modes**: Automatically switches between College and Holiday routines
- **Manual Mode Toggle**: Override automatic mode for special days
- **Analytics Dashboard**: Track completion rates, streaks, and focus hours
- **Local Storage**: All data persists across sessions
- **Responsive Design**: Perfect on mobile and desktop
- **Smooth Animations**: Satisfying task completion interactions

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown (typically http://localhost:5173)

## Usage

- **Check off tasks**: Click/tap any task to mark it complete
- **Switch modes**: Use the mode toggle button to manually override the day type
- **View analytics**: Click the chart icon to see your performance metrics
- **Configure holidays**: Click the settings icon to set your default holiday days

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` folder.
